# select-slot

### Описание {#description}

@short: Срабатывает при выборе слота

### Использование {#usage}

~~~jsx {}
"select-slot": ({
    id: string | number,
    time:[ number, number ]
}) => void;
~~~

### Параметры {#parameters}

Колбэк события `select-slot` принимает объект со следующими параметрами:

- `id` - (обязательный) идентификатор карточки, которой принадлежит выбранный слот
- `time` - (обязательный) массив с временем начала слота в миллисекундах и продолжительностью слота в минутах (время начала задаётся в миллисекундах и представляет местное время)

### Пример {#example}

~~~jsx {7-10}
// создание Booking
const widget = new booking.Booking("#root", {
    data,
    // другие параметры конфигурации
});

// вывод идентификатора выбранного слота
widget.api.on("select-slot", (obj) => {
    console.log(obj.id);
});
~~~
