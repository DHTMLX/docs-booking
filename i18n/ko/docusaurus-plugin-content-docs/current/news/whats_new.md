# 새로운 기능 {#whats-new}

## 버전 1.1 {#version-11}

2025년 3월 12일 출시

### 새로운 기능 {#new-functionality}

- [`renderType`](api/config/booking-rendertype.md) 속성을 통한 카드 지연 렌더링
- [`cardTemplate`](api/config/booking-cardtemplate.md) 및 [`infoTemplate`](api/config/booking-infotemplate.md) 속성을 통한 템플릿 삽입 기능
- [프레임워크 통합](/category/backend-and-frameworks-integration/) 및 [DHTMLX Event Calendar](guides/integration-with-eventcalendar.md) 통합

## 버전 1.0.1 {#version-101}

2024년 11월 29일 출시

### 수정 사항 {#fixes}

- 비데스크톱 레이아웃에서 시작할 때 UI가 깜빡이는 문제
- 모달 창에서 "뒤로" 아이콘의 위치가 잘못 표시되는 문제
- 예약 폼에서 현금 아이콘이 올바르지 않게 표시되는 문제

## 버전 1.0 {#version-10}

2024년 11월 6일 출시

### 초기 기능 {#initial-functionality}

- 항목 슬롯 정의를 위한 [다양한 규칙](guides/configuration.md#define-slot-rules)
- 데이터 항목을 표시하기 위한 구성 가능한 카드
- 날짜, 시간 및 텍스트별 구성 가능한 필터링
- 구성 가능한 예약 대화 상자
- 현지화
