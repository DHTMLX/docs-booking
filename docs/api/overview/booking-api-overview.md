# API overview

## Booking constructor

~~~jsx {}
new booking.Booking("#root", {
	// initial configuration parameters
});
~~~

**Parameters**:

- an HTML container (the ID of the HTML container)
- an object of the configuration parameters ([check here](#booking-properties))
